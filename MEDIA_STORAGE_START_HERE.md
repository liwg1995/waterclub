## 🚀 快速概览

### 这个系统是做什么的？

**媒体文件存储系统**是水立方舞蹈工作室管理系统的文件管理模块，支持：

- ✅ **5种存储方案**：本地存储、腾讯云、阿里云、七牛云、MinIO
- ✅ **后台管理**：图形化的配置和文件管理界面
- ✅ **REST API**：完整的 API 接口，支持上传、查询、删除等
- ✅ **文件链接**：支持复制 URL、HTML、Markdown 三种格式的链接
- ✅ **权限控制**：JWT 认证和权限管理

## 💡 常见问题快速解答

### Q: 系统是否已经完成？
**A:** ✅ 是的，系统已完成 100%。所有 5 种存储方案都已实现，所有核心功能都已测试。

### Q: 生产环境可以使用吗？
**A:** ✅ 是的，系统已通过系统检查，符合生产就绪标准。

### Q: 我需要安装额外的依赖吗？
**A:** ℹ️ 基础功能无需额外依赖。如果要使用云存储，需要安装对应的 SDK（可选）。

### Q: 支持哪些文件类型？
**A:** 系统支持多种文件类型，包括图片、视频、音频、文档等。默认允许 jpg、png、gif、pdf 等常见格式，可配置。

### Q: 文件存储在哪里？
**A:** 
- **本地存储**：`media/uploads/YYYY/MM/DD/filename`
- **云存储**：根据配置，存储在对应的云服务中

### Q: 能否同时使用多个存储配置？
**A:** 可以创建多个配置，但同一时间只有一个配置被激活。可通过一键切换不同的配置。

## 🛠️ 5 分钟快速开始

### 步骤 1：启动服务（1 分钟）
```bash
cd /Users/liwugang/学习/qoder-workspace
python manage.py runserver
```

### 步骤 2：进入后台（1 分钟）
- 打开浏览器：`http://localhost:8000/admin/`
- 用管理员账号登录

### 步骤 3：创建存储配置（2 分钟）
1. 进入：**媒体管理** → **存储配置**
2. 点击"**添加存储配置**"
3. 填写以下信息：
   - 配置名称：`本地存储`
   - 存储类型：`本地存储`
   - 访问URL：`http://localhost:8000/media`
   - 是否启用：✓ 勾选
4. 点击"**保存**"

### 步骤 4：完成！(1 分钟)
现在你可以：
- 进入"媒体文件"查看已上传的文件
- 通过 API 上传新文件
- 在编辑器中使用图片

详细步骤👉 [快速入门指南](./MEDIA_STORAGE_QUICKSTART.md)
